# Chatbot - GPT-3.5 Powered Conversational AI

Welcome to the **Chatbot** wiki! This project is a simple yet powerful chatbot built with [Streamlit](https://streamlit.io/) and [OpenAI's GPT-3.5](https://platform.openai.com/docs/models/gpt-3-5) model.

## Overview

This chatbot template provides a ready-to-use conversational AI interface. It features real-time streaming responses, persistent chat history within a session, and a clean user interface powered by Streamlit's chat components.

## Features

- **Real-time Streaming**: Responses are streamed in real-time as they are generated
- **Chat History**: Messages persist across UI reruns using Streamlit session state
- **Simple Setup**: Only two dependencies required (`streamlit` and `openai`)
- **Secure API Key Input**: API key is entered via a password-masked input field
- **Dev Container Support**: Ready-to-use GitHub Codespaces / Dev Container configuration
